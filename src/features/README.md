# Features

Feature-sliced modules live here. Each feature owns its screens, feature-local
components, and hooks, and composes the shared design system (`src/components`),
types (`src/types`) and services (`src/services`).

Planned feature slices for ZONO:

- `dashboard/` — agent home: opportunities feed, today's agenda, KPIs
- `properties/` — listing management, property journey, distribution
- `buyers/` — buyer CRM, preferences, saved matches
- `sellers/` — seller CRM, exclusivity tracking
- `leads/` — inbound lead funnel and qualification
- `matching/` — buyer↔property matching surface
- `market/` — market areas, trends, events
- `assistant/` — the ZONO AI assistant surface
- `automations/` — no-code workflow builder
- `calendar/` — agenda, viewings, signings
- `documents/` — documents and e-signature flows

Kept empty for now — the foundation phase only prepares structure.
